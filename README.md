# Tanks
Tanks is a HTML5 multiplayer game. This is a work in progress.

## Prerequisites
1. [Nodejs](https://nodejs.org/en/)
2. [Npm](https://www.npmjs.com)

## Running the server
1. Run `npm install`
2. Run `npm start`
3. Connect to the game on `localhost:2000`
